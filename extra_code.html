Canvas power app solution

Integrated request form and approval process

April 1, 2024 Power platform

A team faced issues with processing approvals to grant access to local systems with sensitive data. The existing process was for managers to email whoever they knew by word-of-mouth could grant system access. The person would respond if the new system user was approved for system access and ask any follow-up questions about the specific systems or system environments the user needed. Someone would then be emailed or messaged separately to grant the requested access, then messaging the new user to notify them of their newly granted access. Someone on the team would then manually save the related emails into a SharePoint folder as documentation for audits. The team requested a front-end form interface with approval triggers. A Canvas Power App was paired with a SharePoint List and Power Automate flows to consolidate the workflow.
